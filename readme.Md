## ğŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- PostgreSQL (v13 or higher)
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/RahatInCode/hireMe.git

# Navigate to project directory
cd hireme

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Configure your database connection in .env

# Run database migrations
npm run migrate

# Start the development server
npm run dev
```

The application will be available at `http://localhost:3000`

## ğŸ“– Usage Guide

### For HR Administrators

1. **Add a New Hire**
   - Click "New Employee" button
   - Enter basic details (name, email, role, department, start date)
   - Upload resume (optional - AI will extract relevant information)
   - Click "Start Onboarding"

2. **Customize Workflows**
   - Navigate to "Templates" section
   - Create department-specific checklists
   - Set task dependencies and deadlines
   - Assign responsible parties

3. **Monitor Progress**
   - View real-time dashboard showing all active onboardings
   - Filter by department, status, or date range
   - Click any employee to see detailed progress

### For New Hires

1. **Receive Welcome Email**
   - Automatic email sent on submission with login credentials
   - Personalized message from your manager

2. **Complete Your Profile**
   - Fill in additional details through guided forms
   - Upload required documents
   - E-sign necessary agreements

3. **Access Your Dashboard**
   - View your onboarding checklist
   - Ask the AI assistant any questions
   - Track upcoming training sessions

### For IT Teams

1. **Receive Notifications**
   - Automatic alerts when new accounts are needed
   - Clear specifications for access levels and equipment
   - Integrated with your existing ticketing system

2. **Update Status**
   - Mark tasks complete as you provision accounts
   - System automatically notifies HR and the new hire
